# OSOP: Açık Kaynak & İleri Sızma Final Projesi

Bu proje, **Open Source Operating System Project (OSOP)** kapsamında, Yapay Zeka (AI) sistemlerinin güvenliğini ve manipülasyon risklerini analiz etmek amacıyla geliştirilmiştir.

##  Proje Kimliği
* **Kategori:** AI & Web Security
* **Konu:** Large Language Model (LLM) Prompt Injection & Jailbreak Analizi
* **Öğrenci:** <Burhan Yavaş>
* **Öğrenci No:** <2420191018
* **Ders:** İleri Sızma Testi

##  Proje Amacı
Modern LLM sistemlerinin (ChatGPT, Gemini, Llama vb.) güvenlik açıklarını tespit etmek, "Prompt Injection" saldırı vektörlerini simüle etmek ve bu saldırılara karşı açık kaynaklı savunma stratejileri geliştirmektir.

##  Depo Yapısı
* **`/docs`**: Proje dökümantasyonu ve analiz raporları.
* **`/researchs`**: Literatür taraması ve zafiyet veritabanı incelemeleri.
* **`/specs`**: Teknik gereksinimler ve saldırı senaryoları.
* **`/src`**: POC (Proof of Concept) kodları ve test scriptleri.

##  Kurulum ve Kullanım
Bu proje Python tabanlı test araçları içermektedir. Detaylar `src` klasöründedir.
